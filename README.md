# 🤖 PDF ChatBot — AI Assistant

A ChatGPT-style chatbot that lets you upload a PDF and ask questions using AI.

## Features
- 📄 PDF Upload (drag & drop)
- 💬 ChatGPT-style chat interface
- 🧠 Powered by Groq LLaMA 3.1 (Free AI)
- 📜 Chat history
- 🐍 Python Flask backend

## Tech Stack
- Python + Flask
- Groq API (LLaMA 3.1)
- HTML, CSS, JavaScript
- PyMuPDF

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key in `.env`
4. Run: `python app.py`
5. Open: `http://localhost:5000`
